Enviar Código Fonte - Aula 01

Antes de seguir para a próxima aula, você deve fazer upload do seu código-fonte para atender ao execício desta aula.

Orientações para envio do código-fonte:

  1. Clique em "Enviar Código-Fonte" para abrir a janela de seleção de arquivos;
  2. Navegue até seu diretório de projetos e abra a pasta chamada “Prática 01”;
  3. Selecione todos os arquivos que correspondem ao código-fonte deste exercício, incluindo HTML, CSS, Javascript e Imagens, se aplicável;
  4. Tipos de arquivos aceitos para envio: HTML, CSS, Javascript e Imagens

ATENÇÃO: Após enviar os arquivos, aguarde a confirmação de recebimento para sair desta página
